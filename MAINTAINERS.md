## 如何成为维护人员

我们欢迎有经验和热情的贡献者加入维护团队。要成为维护人员，你应该：

### 要求

- ✅ 至少提交 5 个有意义的 PR
- ✅ 代码质量高，遵循项目规范
- ✅ 积极参与 Issue 讨论和代码审查
- ✅ 遵守行为准则
- ✅ 承诺定期投入时间维护项目

### 流程

1. **表达兴趣**
   - 在 Issue 或 Discussion 中表达你的兴趣
   - 或发送邮件至 [solitaryhao8@gmail.com](mailto:solitaryhao8@gmail.com)

2. **核心团队评估**
   - 审查你的贡献历史
   - 评估你的技能和承诺
   - 进行讨论和面试

3. **正式邀请**
   - 如果通过评估，将收到正式邀请
   - 讨论具体的维护职责
   - 签署维护人员协议

4. **入职**
   - 获得仓库权限
   - 加入维护人员团队
   - 在 README 中获得致谢

所有维护人员应该：

- 🤝 尊重所有贡献者和用户

- 🔒 保护项目安全
- 🎯 专注于项目目标
- 🌱 帮助新贡献者成长
- 🚀 推动项目向前发展
